## 背景知识

+ 二进制加法：根据十进制加法理解二进制加法；
+ 正数/负数的二进制表示：
	- 二进制编码中，负数首位为1，正数为0；
	- -x的二进制码为：x的二进制码所有位取反，再加1；
+ 一些算数公式的推导与理解（假定：~为取反操作）：
	- -1 = ~0
	- 1 = ~(~0 + ~0)
	- -x = ~(x + ~0)
	- x+1 = ~(~x + ~0)
	- x-y = x + (-y) = ~(~x + y) = ...
	
	比如：-1 = ~1 + 1
		- ⇔ 以4位二进制来讲，~1二进制码为1110，1二进制码为0001，故~1 + 1的二进制码为1111，即~0；
