# notes.nand2tetris

nand2tetris，又名“从与非门到俄罗斯方块”，手把手的教你从基础的与非门出发，一步一步构建现代计算机的硬件平台（hardware platform）和软件体系（software hierarchy），对理解底层计算机原理很有裨益。设想一下，用你自己亲手设计的编程语言编写程序，然后运行在你自己亲手设计的硬件平台上......这里为本人的学习记录，毕竟好记性不如烂笔头。
